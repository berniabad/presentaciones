# Reporte Kaiyi Evento

Reporte del evento de lanzamiento de Kaiyi en Argentina.

## Estructura

- `index.html`: presentacion web final.
- `styles.css`: sistema visual y estilos responsive.
- `script.js`: interacciones, navegacion y animaciones.
- `estructura-reporte.md`: guion editorial, datos pendientes y assets necesarios.
- `assets/img`: imagenes finales optimizadas para la presentacion.
- `assets/logos`: logos de Kaiyi, Gloaming y aliados.
- `assets/fonts`: tipografias locales usadas por la pieza.
